## 导言

---
这是一个用来更加便捷编写我的世界spigot插件的开发工具，这个工具可以将复杂的Java代码开发工具转变为简单的连连看，也就是节点编辑器。
当然这不仅仅是个节点编辑器，里面还将会集成很多功能，这些功能是选择性的，因此不用担心存储问题。
这个项目完全开源。
